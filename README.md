# がんばる応援カレンダー

目標をタイトルに入力してカレンダーを作成。  
日付の数字部分をクリックすると、日付部分の背景色が変わって褒められます。  
目標継続のサポートをするスタンプカードのようなものです。  
イメージはラジオ体操のカードです。
